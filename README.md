# Sheet

独自スプシのプロトタイプ

<br/><br/>

## 実装概要

- react-hook-form を利用して cell の値を handling
- （react-hook-form の useFieldArray を用いて動的な行の追加・削除も実装可能）

<br/><br/>

## 機能

- <input/> が table 形式で並んでいること
- onBlur で submit できること
- submit 時に table の全ての内容を取得して post できること

<br/><br/>
